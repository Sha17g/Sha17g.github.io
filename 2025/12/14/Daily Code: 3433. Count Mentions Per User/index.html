品酒图鉴：莫吉托
这可能是我喝过的最好喝的酒。45ml白朗姆，20ml青柠汁，6片薄荷叶，2小勺砂糖，苏打水补满，这是IBA给的标准做法，很简单的配方，但入口的清爽超乎意料，做法感觉有点像自由古巴，但是没有了自由古巴中的药味，会不会自由古巴本来的味道差不多清爽，只不过简化焦糖为可口可乐后的版本中可口可乐的风味物质影响了酒体的味道？ 这个酒真的非常推荐，感觉比大多数不含酒精的饮料还好喝。 在网上买了一小盆薄荷，但是怎么看都像是老家路边的十香，搜了搜发现国内现在十香因为好种，好栽，味道相似，已经逐渐“抢夺”掉薄荷的名字了。以至于真正的薄荷，现在反而被称为“野薄荷”，虽然昨晚调出来的效果不错，但是不是应该买一盆“野薄荷”用来调酒呢？
